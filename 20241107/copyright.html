序言与致谢：为何使用这一套代码？

此项目基于我 2021 年时使用 Hielo 开源框架主题所做的服务器官网改动而来。
那个时候还没有深入地接触到样式表、JavaScript、HTML 这些系统性的东西，我当时能做到的只是通过右键元素审查的方法进行替换（笑
这次大作业之所以选用一套曾用过的框架主题，一方面是通过已学的知识尝试去梳理对应的 JavaScript 和样式表去理解它们在网页中到底发挥了什么作用，另一方面也是对过去的我，一个什么都不懂的学生但只是为了修改而修改能熬下来的那份纯粹的热情的一份坚守与传承。
但光有热情是不够的，最重要的是要有知识来支撑，这样我们的能力才能获得提升。
在此也要特别感谢赵老师，感谢她能够为我们非计算机专业的学生开这样的一门课程，让我能有机会从更加接近前端开发的角度，去深刻体会网页设计的精巧与深入。
现在来看，我对这一套源代码的理解虽然谈不上和专攻 JavaScript 的同志同等水平，但至少我能读懂，并且能根据自己的需要直接修改 JavaScript 代码了，我想这也是一种进步吧。（笑
